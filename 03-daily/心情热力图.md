# 心情热力图

说明：
- 在每天的日记属性里填写 `mood`。
- 可选值：`开心`、`一般`、`不开心`、`重要`。
- 颜色：粉色=开心，蓝色=一般，黄色=不开心，紫色=重要。
- 想看别的年份，就修改本文件最上方的 `year` 属性。

## 今年

```dataviewjs
const year = Number(dv.current().year ?? new Date().getFullYear());

const moodMap = {
  "开心": { color: "happy", label: "开心" },
  "一般": { color: "normal", label: "一般" },
  "不开心": { color: "sad", label: "不开心" },
  "重要": { color: "important", label: "重要的事情记录" },
  "重要的事情记录": { color: "important", label: "重要的事情记录" }
};

const calendarData = {
  year,
  colors: {
    happy: ["#f8c8dc", "#f5a9c8", "#f17fb0", "#ec5799", "#d93682"],
    normal: ["#cfe3ff", "#acd0ff", "#83b8ff", "#5a9fff", "#347fe0"],
    sad: ["#fff4b0", "#ffeb7a", "#ffe04a", "#ffd21f", "#e6b800"],
    important: ["#e0ccff", "#caa9ff", "#b383ff", "#9a5cff", "#7c3fd1"]
  },
  showCurrentDayBorder: true,
  defaultEntryIntensity: 4,
  intensityScaleStart: 1,
  intensityScaleEnd: 5,
  entries: []
};

for (const page of dv.pages('"03-daily/daily"').where(p => p.mood)) {
  const date = page.file.name;
  const mood = Array.isArray(page.mood) ? String(page.mood[0]).trim() : String(page.mood).trim();
  const item = moodMap[mood];

  if (!date || !item) continue;

  calendarData.entries.push({
    date,
    color: item.color,
    intensity: 5,
    content: await dv.span(`[[${page.file.path}|${date} ${item.label}]]`)
  });
}

renderHeatmapCalendar(this.container, calendarData);
```
